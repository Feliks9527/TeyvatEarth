# 提瓦特地球 · Teyvat Earth

原神风格的交互式 3D 地球。基于真实地理数据渲染成插画风地图，叠加七天神像、传送锚点与秘境，点击锚点可传送并「进入当地」查看 360° 全景。

纯前端，无后端依赖。

## 特性

- **插画风地球**：Natural Earth 110m 真实国界数据，自定义 GLSL 卡通渲染（色阶光照 + 边缘光），Perlin 噪声生成的生物群系地形着色（冰原 / 针叶林 / 森林 / 草原 / 沙漠 / 雨林）
- **三套贴图风格**：提瓦特 / 黎明 / 深渊，一键切换
- **百余个锚点**：
  - ⛩ 七天神像 — 各国首都
  - ✦ 传送锚点 — 六大洲重要城市
  - ❖ 秘境 — 珠峰、撒哈拉之眼、大堡礁等自然奇观
- **进入当地**：传送过场后进入可拖拽环视的 360° 全景，地点代表性照片取自维基百科，按提瓦特七国元素属性（风/岩/雷/草/水/火/冰）着色，并显示地区档案（元素 / 天气 / 地貌 / 风味文案）
- **HYWenHei-85W** 字体全局应用（原神官方汉仪文黑）

## 技术栈

Vite 5 · Three.js 0.169 · topojson-client · 纯 Canvas/GLSL 程序化美术

## 本地运行

```bash
npm install
npm run dev      # http://localhost:5180
npm run build    # 产物输出到 dist/
```

## 数据与资源

- 地理数据：[Natural Earth](https://www.naturalearthdata.com/)（110m，公共领域）
- 地点照片：[维基百科](https://www.wikipedia.org/) pageimages 接口（运行时拉取）
- 字体：HYWenHei-85W（汉仪文黑，原神所用商业字体，仅供学习演示）

## 操作

拖拽旋转 · 滚轮缩放 · 点击锚点开启传送 · 卡片内「进入当地」查看全景
